You can contribute!

Join the WPI Global Lab and the Global Humanitarian Alliance next Friday (11/30) and Saturday (12/1) to learn how you can help support victims of Venezuela's humanitarian crisis. You don't need programming experience to contribute to this event (see below). If you are a programmer, join us and learn new skills as you code for a cause! No matter your background, HackForVenezuela has something for you.


"What if I don't know how to make an app?"

Awesome! Learn to build an Android app with Java while contributing to an important cause at our Android workshop Saturday morning. Not a programmer? Attend our Mendix workshop and learn to to build an app without any code!


"What if I don't want to make an app?"

You don't have to! GHA will be running workshops, talks, and discussions throughout the day so you can learn about different aspects of Venezuela's escalating humanitarian crisis, and what you can do to help. Find a way to contribute that fits your skills and interests!


"What if I'm not free the whole weekend?"

While we encourage you to join us for as much of the event as possible, you don't need to be in Foisie the whole time! You can drop in for any of the talks/workshops, and see what teams are working on. Just be sure to register!


Questions? Comments? Ideas?

Don't hesitate to contact us at gr-hfv-questions@wpi.edu! We want this event to be productive and enjoyable for everyone involved.

Prizes, guest speakers and mentors will be announced by Tuesday Nov. 27th so stay tuned!
